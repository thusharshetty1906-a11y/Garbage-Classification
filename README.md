# Garbage Classification using Deep Learning

## Project Overview
This project classifies garbage images into different categories using Deep Learning with TensorFlow and Keras.

## Dataset
The dataset contains images of:
- Cardboard
- Glass
- Metal
- Paper
- Plastic
- Trash

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Google Colab

## Model
A Convolutional Neural Network (CNN) is used for image classification.

## Features
- Image preprocessing
- Data augmentation
- CNN model training
- Model evaluation
- Garbage prediction

## How to Run

pip install tensorflow numpy matplotlib
